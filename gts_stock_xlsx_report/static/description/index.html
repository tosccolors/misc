Real Time Stock XLSX Report

This module allows to get stock report in xlsx file for any location or warehouse or date range

Features

Get the output in a xlsx file listing Product name, Internal Reference, location name, Opening, Incoming, Outgoing, Adjustment, Internal Transfers, closing stock

Option to filter by Product, location, warehouse

Multi Company Specific Inventory Report

Generate Stock Inventory Real Time Report in XLSX by specific dates

Option to skip products / Locations having stock 0

Option for fetching Valuation also
